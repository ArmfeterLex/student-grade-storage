# Student Grade Storage

Приложение на C# Windows Forms для сохранения и чтения информации об учениках.

## Функциональность

### 📝 Ввод данных
- Имя ученика (текстовое поле)
- Класс (RadioButton: 9, 10, 11)
- Предмет (ComboBox с выбором)

### 💾 Сохранение
- Запись данных в файл sam3.txt
- Формат: "Имя,Класс,Предмет"
- Проверка заполнения всех полей

### 📂 Чтение
- Загрузка данных из файла sam3.txt
- Отображение в MessageBox
- Обработка ошибок (файл не найден)

### 🔄 Очистка
- Кнопка отмены очищает все поля

## Интерфейс

- txtName — поле для имени
- radioButton9,10,11 — выбор класса
- comboBoxSubject — выбор предмета
- btnSave — сохранить данные
- btnRead — прочитать данные
- btnCancel — очистить поля

## Пример работы

**Сохранение:**
Ввод: "Иванов", класс 10, предмет "Математика"
→ файл sam3.txt: "Иванов,10,Математика"

**Чтение:**
MessageBox показывает: "Данные:\nИванов,10,Математика"

## Куда можно встроить

- В системы учёта учащихся
- В электронные журналы
- В демонстрацию работы с файлами
- В примеры валидации ввода

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При сохранении файл перезаписывается (только одна запись). Для хранения нескольких записей требуется модификация кода (добавление в конец).
